# 快手 & Bilibili 解析下载工具
> 基于 FFmpeg 的视频解析工具，支持快手视频 $ 图片去水印、Bilibili 视频/音频分离下载，支持 M4A 转 MP3 格式转换

## 功能特点
1.  **快手视频处理**：解析快手短视频并去除水印，直接下载无水印视频文件
2.  **Bilibili 资源分离**：单独下载 B 站视频的**视频流**和**音频流**，满足不同场景需求
3.  **音频格式转换**：可将下载的 M4A 音频文件转换为通用的 MP3 格式
4.  **基于 FFmpeg**：依托 FFmpeg 强大的音视频编解码能力，保证处理速度和兼容性

## 使用教程
### 1. 快手视频去水印下载
#### 步骤 1：获取快手视频分享链接
打开快手 App → 选择目标视频 → 点击**分享** → 复制**视频链接**

- 输出目录下生成无水印视频文件（默认 m4a 格式）

### 2. Bilibili 视频/音频分离下载

## 常见问题
### Q1：解析失败提示「链接无效」？
A1：
- 检查视频链接是否正确，是否包含 `https://` 前缀
- 确认视频是否为公开可见状态，私密视频无法解析

### Q2：FFmpeg 命令执行报错？
A2：
- 检查 FFmpeg 是否正确配置到环境变量
- 确认输入文件路径是否存在中文或特殊字符，建议使用英文路径
